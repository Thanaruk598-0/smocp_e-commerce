# Student Uniform Sales and Order Management System
Web Application Project
This project is a web-based system designed to manage student uniform sales and orders.
The system helps streamline product management, order processing, and sales reporting
for both administrators and users.

## Project Objectives
- To manage student uniform products and orders efficiently
- To support sales tracking and reporting
- To reduce manual record-keeping and improve data accuracy

## System Features

### User
- Browse available uniform products
- Place uniform orders
- View order history

### Admin
- Manage products and inventory
- Process and track orders
- Generate sales reports and analytics
